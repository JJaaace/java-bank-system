# Java Bank System

A simple console-based banking application built in Java that allows users to create accounts, deposit and withdraw money, and view account details.

## Features
- Create accounts with a username and password
- Deposit money into an account
- Withdraw money with balance validation
- View a specific account's balance
- View all accounts
- Basic input validation and error handling

## Technologies Used
- Java
- ArrayList
- Scanner (for user input)

## How to Run
1. Compile the program:
   javac BankSystem.java
2. Run the program:
   java BankSystem

## Notes
This project was created as a beginner-level Java program to practice working with user input, loops, conditionals, and data structures.

Future improvements could include:
- Using classes instead of multiple ArrayLists
- Adding a login system
- Saving account data to a file
